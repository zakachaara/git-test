# Exercise 1 — Configure Git

Before making any commits, Git needs to know who you are.

## Your Task

Inside your practice Git repository, set two **local** config values:

    git config user.name  "Your Name"
    git config user.email "you@example.com"

Replace the placeholder values with any non-empty name and email address.

## Verify

When you are ready, run:

    git-test verify

## Hints

- `git config --list --local` shows all settings in the local `.git/config`.
- You do **not** need to make a commit for this exercise.
